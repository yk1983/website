정의

a 요소는 텍스트에 링크를 설정하고자 할 때 사용한다.

속성

href : 링크 주소를 지정한다.

target : 링크를 어떤 프레임에 열것인지를 결정한다.

  • _blank : 새창에 열기
  • _parent : 부모창에 열기
  • _self : 자신의 창에 열기
  • _top : 화면이 프레임단위로 이루어져 있는 경우 top 프레임에 열기
  • [name] : 화면이 프레임단위로 이러어져 있는 경우 지정된 name에 열기

설명

a 요소는 문단, 목록, 테이블뿐만 아니라 섹션 전체를 감쌀 수도 있다.

a 요소안에 버튼이나 링크등 인터렉티브한 요소가 포함되지 않도록 한다.

링크의 상태

스타일을 정의할 때 링크 상태들은 순서대로 정의 해야 의도한 대로 스타일을 적용할 수 있다. active를 hover보다 먼저 정의 했다면 hover가 active를 오버라이드 하여 active상태를 볼 수가 없게 된다.

  • link : 기본적인 링크 상태로 아직 방문한적이 없거나 아무런 행동도 하기전인 상태
  • visited : 방문한 적이 있는 링크의 상태
  • hover : 마우스 커서가 링크위에 올라가 있는 상태
  • focus : 링크에 키보드등을 이용해서 포커스가 위치해있는 상태
  • active : 마우스를 눌렀다가 놓는 동안의 상태
