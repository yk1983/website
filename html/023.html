정의

colgroup요소는 서식 지정을 위해 표의 하나 이상의 열 그룹을 지정합니다.

colgroup요소는 태그는 각 행에 대해 각 셀의 스타일을 반복하는 대신 전체 열에 스타일을 적용하는 데 유용합니다.

지원 브라우저

속성

Attribute Description
align HTML5에서는 지원되지 않습니다. 열 그룹의 내용을 정렬합니다.
char HTML5에서는 지원되지 않습니다. 열 그룹의 내용을 문자에 정렬합니다.
charoff HTML5에서는 지원되지 않습니다. 내용이 char속성으로 지정된 문자에서 정렬 될 문자 수를 설정합니다.
span 열 그룹이 확장되어야하는 열의 수를 지정합니다.
valign HTML5에서는 지원되지 않습니다. 열 그룹의 내용을 세로로 정렬합니다.
width HTML5에서는 지원되지 않습니다. 열 그룹의 너비를 지정합니다.

설명

표의 열을 구조적인 그룹화를 위한 요소입니다.

colgroup 요소는 col 요소를 포함할 수 있습니다.

참고 : colgroup 요소는 caption 요소의 다음에 thead, tbody, tfoot 및 tr 요소 앞에 table 요소의 하위 항목이어야합니다.

팁 : colgroup 내의 열에 다른 속성을 정의하려면 colgroup 요소에서 col 요소를 사용하십시오.

기본 CSS 설정

colgroup { 
    display: table-column-group;
}

예제

<table>
    <colgroup>
        <col span="2" style="background-color:red">
        <col style="background-color:yellow">
    </colgroup>
    <tr>
        <th>ISBN</th>
        <th>Title</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>3476896</td>
        <td>My first HTML</td>
        <td>$53</td>
    </tr>
</table>
