정의

command요소는 사용자가 호출 할 수 있는 명령을 지정하는데 사용하는 태그입니다.

지원 브라우저

Not supported

속성

Attribute Description
Icon 명령을 나타내는 그림을 지정하는 속성입니다.
label 명령의 이름을 지정하는 속성입니다. 이 속성은 반드시 명시되어야 합니다.
type 명령의 종류를 지정합니다.
  • command : 연합된 액션을 갖는 일반적인 명령을 나타냅니다.
  • checkbox : 토글이 되는 옵션이나 상태를 나타냅니다.
  • radio : 목록중에서 하나의 아이템의 선택을 나타냅니다.
disabled 불리언 속성으로 커맨드가 현재상태에서 사용할 수 없음을 나타냅니다.
checked 불리언 속성으로 커맨드가 선택되었음을 나타냅니다. type속성이 checkbox, radio상태가 아니면 생략되어야만 합니다.
laberadiogroupl 커맨드 자신이 토글되었을 때 토글된 커맨드 그룹에 이름이 주어집니다. type속성이 radio일 때만 사용할 수 있습니다.

설명

<command>태그는 라디오 버튼, 체크 박스 같은 명령 단추를 재설정 하며, 반드시 <menu> 태그 안에서 있어야 합니다.

참고 : 주로 사용되고 있는 브라우저 중 해당 태그를 지원하는 브라우저가 아직 없습니다.

command요소는 HTML5에서 새로 추가된 요소이지만 브라우저가 지원하지 않았고 결국 삭제되었습니다. 대신 menuitem요소를 사용하기를 권장합니다.

기본 CSS 설정

예제

<menu type="toolbar">
    <command type="radio" radiogroup="alignment" checked="checked"
    label="Left" icon="icons/alL.png" onclick="setAlign('left')">
    <command type="radio" radiogroup="alignment"
    label="Center" icon="icons/alC.png" onclick="setAlign('center')">
    <command type="radio" radiogroup="alignment"
    label="Right" icon="icons/alR.png" onclick="setAlign('right')">
    <hr>
    <command type="command" disabled
    label="Publish" icon="icons/pub.png" onclick="publish()">
</menu>

<command type="command|checkbox|radio">
<menu>
    <command type="command" label="Save" onclick="save()">Save</command>           
</menu>
