정의

figure요소는 일러스트, 다이어그램, 사진, 코드등에 주석을 다는 용도로 사용됩니다.

지원 브라우저

6.0 9.0 4.0 5.0 11.1

설명

일러스트레이션, 다이어그램, 사진, 코드 목록 등과 같은 자체 포함 된 콘텐츠를 지정합니다.

이 요소가 제거 되더라도 문서의 주된 흐름에는 큰 영향을 미치지 않습니다. 즉, 요소의 내용은 주 흐름과 관련이 있지만 위치는 주 흐름과 관계가 없으며 제거 된 경우 문서의 흐름에 영향을 미치지 않아야합니다.

기본 CSS 설정

figure { 
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 40px;
    margin-right: 40px;
}

예제

문서에서 사진을 마크 업 :
<figure>
    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>
