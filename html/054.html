정의

img요소는 이미지를 삽입합니다.

지원 브라우저

설명

src속성은 필수속성입니다.

img요소는 src와 alt의 두 가지 필수 속성이 있습니다. (alt속성은 반드시 사용하는 것이 좋습니다. 그 값 또한 비어있지 않고, 이미지를 대체하기에 적합한 텍스트를 마크업 해야 합니다.)

참고 : 이미지는 기술적으로 HTML 페이지에 삽입되지 않으며 이미지는 HTML 페이지에 링크됩니다. img 요소는 참조 된 이미지를위한 공간을 생성합니다.

팁 : 이미지를 다른 문서에 연결하려면 a요소 안에 img요소를 중첩하면됩니다.

HTML5에서는 align, border, hspace 및 vspace 속성이 지원되지 않습니다.

HTML에서 닫는 태그가 없습니다.

XHTML에서는 태그가 제대로 닫혀 있어야합니다.

속성

Attribute Description
align 주변 요소에 따라 이미지의 정렬을 지정합니다. HTML5에서는 지원되지 않습니다.
alt 이미지의 대체 텍스트를 지정합니다.
border 이미지 주변의 테두리 너비를 지정합니다. HTML5에서는 지원되지 않습니다.
crossorigin 크로스 원본 액세스가 캔버스와 함께 사용될 수 있는 타사 사이트의 이미지를 허용합니다.
height 이미지의 높이를 지정합니다.
width 이미지의 너비를 지정합니다.
hspace 이미지의 왼쪽과 오른쪽에있는 공백을 지정합니다. HTML5에서는 지원되지 않습니다.
ismap 이미지를 서버 측 이미지 맵으로 지정합니다.
longdesc 이미지의 상세한 설명에 대한 URL을 지정합니다.
sizes 페이지 레이아웃마다 이미지 크기를 지정합니다.
usemap 이미지를 클라이언트 측 이미지 맵으로 지정합니다.
vspace 이미지의 위 아래 에있는 공백을 지정합니다. HTML5에서는 지원되지 않습니다.
src 이미지의 URL을 지정합니다.
srcset 다른 상황에서 사용할 이미지의 URL을 지정합니다.

기본 CSS 설정

img { 
    display: inline-block;
}

예제

<img src="smiley.gif" alt="Smiley face" height="42" width="42">
