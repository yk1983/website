정의

code요소는 구문 태그입니다. 이 요소는 컴퓨터 코드를 정의합니다.

지원 브라우저

설명

팁 : 이 태그는 더 이상 사용되지 않지만 CSS를 사용하면 더 많은 효과를 얻을 수 있습니다.

XML요소 이름, 파일 이름, 컴퓨터 프로그램등 컴퓨터가 인식할수 있는 일련의 컴퓨터 코드를 나타냅니다.

pre요소와 함께 사용하여 코드블럭을 마크업 할 수 있습니다.

기본 CSS 설정

code {
    font-family: monospace;
}

예제

<code>A piece of computer code</code>

<pre>
    <code class="language-pascal">
        var i: Integer;
        begin
        i := 1;
        end.
    </code>
</pre>
