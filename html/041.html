정의

font요소는 글꼴의 글꼴, 글꼴 크기 및 텍스트 색상을 지정합니다.

HTML5에서 지원되지 않습니다. 대신 CSS를 사용합니다.

지원 브라우저

설명

속성

Attribute Description
color 텍스트의 색을 지정합니다. HTML5에서는 지원되지 않습니다.
face 텍스트의 글꼴을 지정합니다. HTML5에서는 지원되지 않습니다.
size 텍스트의 크기를 지정합니다. HTML5에서는 지원되지 않습니다.

예제

<ont size="3" color="red">This is some text!<font>
<ont size="2" color="blue">This is some text!<font>
<ont face="verdana" color="green">This is some text!<font>
