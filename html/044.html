정의

frame요소는 frameset내의 하나의 특정 창(프레임)을 정의합니다.

HTML5에서 지원되지 않습니다.

지원 브라우저

속성

Attribute Description
frameborder 프레임 주위에 테두리를 표시할지 여부를 지정합니다. HTML5에서는 지원되지 않습니다.
longdesc 프레임 내용에 대한 자세한 설명이 포함 된 페이지를 지정합니다. HTML5에서는 지원되지 않습니다.
marginheight 프레임의 위쪽 및 아래쪽 여백을 지정합니다. HTML5에서는 지원되지 않습니다.
marginwidth 프레임의 왼쪽 및 오른쪽 여백을 지정합니다. HTML5에서는 지원되지 않습니다.
name 프레임의 이름을 지정합니다. HTML5에서는 지원되지 않습니다.
noresize 프레임의 크기를 재조정 할 수 없도록 지정합니다. HTML5에서는 지원되지 않습니다.
scrolling 프레임에 스크롤 막대를 표시할지 여부를 지정합니다. HTML5에서는 지원되지 않습니다.
src 프레임에 표시 할 문서의 URL을 지정합니다. HTML5에서는 지원되지 않습니다.

설명

frameset의 각 frame은 테두리, 스크롤링, 크기 조정 등의 다양한 속성을 가질 수 있습니다.

참고 : 프레임이 포함 된 페이지의 유효성을 검사하려면 <!DOCTYPE> 이 "HTML Frameset DTD"또는 "XHTML Frameset DTD"로 설정되어 있는지 확인하십시오.

HTML에서 frame요소에는 끝 태그가 없습니다. XHTML에서는 frame태그가 제대로 닫혀 있어야합니다.

예제

<frameset cols="25%,50%,25%">
    <frame src="frame_a.htm">
    <frame src="frame_b.htm">
    <frame src="frame_c.htm">
</frameset>
