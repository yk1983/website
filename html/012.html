정의

bdo는 Bi-Directional Override를 나타냅니다.

bdo요소는 텍스트의 방향을 명시적으로 지정합니다.

지원 브라우저

속성

Attribute Description
dir 필수 사항. <bdo> 요소 안의 텍스트의 텍스트 방향을 지정합니다.
  • ltr(left to right) : 왼쪽에서 오른쪽으로 출력
  • rtl(right to left) : 오른쪽에서 왼쪽으로 출력

설명

dir 속성을 반드시 사용해야 합니다.

기본 CSS 설정

bdo { 
    unicode-bidi: bidi-override;
}

예제

<bdo dir="rtl">
This text will go right-to-left.
</bdo>
