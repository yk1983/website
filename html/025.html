정의

data요소는 주어진 내용을 기계가 읽을 수있는 번역으로 연결합니다.

지원 브라우저

N/S

속성

Attribute Description
value 요소 내용의 기계 가독성 변환을 지정합니다.

설명

<data> 태그는 주어진 내용을 기계가 읽을 수있는 번역으로 연결합니다.

팁 : 내용이 시간 또는 날짜와 관련된 경우 time 요소를 대신 사용해야합니다.

command요소는 HTML5에서 새로 추가된 요소이지만 브라우저가 지원하지 않았고 결국 삭제되었습니다. 대신 menuitem요소를 사용하기를 권장합니다.
