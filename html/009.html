정의

b요소는 텍스트를 굵게 표시된 텍스트를 지정합니다.

속성

설명

참고 : HTML 5 사양에 따르면 <b> 태그는 다른 태그가 더 적합하지 않을 경우 마지막 수단으로 사용해야합니다. HTML 5 사양에서는 표제를 <h1> ~ <h6> 태그로 표시하고 강조 텍스트는 <em> 태그로 표시하고 중요한 텍스트는 <strong> 태그로 강조 표시한 텍스트는 <mark> 태그를 사용하여 표시해야한다고 명시되어 있습니다.

팁 : CSS font-weight속성을 사용하여 굵은 텍스트를 설정할 수도 있습니다.

기본 CSS 설정

b { 
    font-weight: bold;
}

예제

This is normal text - <b>and this is bold text</b>.
