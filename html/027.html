정의

dd요소는 설명 목록의 용어 / 이름을 설명하는 데 사용됩니다.

dd요소는 dl요소의 용어-설명 그룹에서 설명, 정의, 값을 나타냅니다.

지원 브라우저

속성

설명

dl요소의 하부 요소입니다. 정의된 용어(dt요소)에 대한 설명을 지정합니다.

dd요소는 dl(설명 목록 정의)요소 및 dt(용어 / 이름 정의)요소와 함께 사용됩니다.

dd요소 안에 단락, 줄 바꿈, 이미지, 링크, 목록 등을 넣을 수 있습니다.

HTML 4.01에서 정의 목록의 항목을 설명하는 데 사용되었습니다. HTML5에는 설명 목록의 용어 / 이름을 설명하는 데 사용됩니다.

기본 CSS 설정

dd { 
    display: block;
    margin-left: 40px;
}

예제

<dl>
    <dt>Coffee</dt>
    <dd>Black hot drink</dd>
    <dt>Milk</dt>
    <dd>White cold drink</dd>
</dl>
