정의

div요소는 태그자체는 특별한 의미가 없으며 문서의 영역을 지정하는데 사용합니다.

div요소는 HTML문서에서 부분 또는 섹션을 정의합니다.

지원 브라우저

속성

Attribute Description
align 요소 내에서 내용의 정렬을 지정합니다. HTML5에서는 지원되지 않습니다. CSS를 이용하세요.

설명

id속성이나 class속성을 이용하여 식별자를 지정하며 종종 CSS로 스타일을 지정하거나 JavaScript로 특정 작업을 수행하기 위해 다른 HTML 요소의 컨테이너로 사용됩니다. 이러한 식별자로 CSS를 적용합니다.

div요소는 다른 적절한 요소가 없는 경우 마지막 방편으로 사용하기를 강력히 권장합니다. 그렇지 않으면 접근성을 어렵게 할뿐만 아니라 관리의 어려움을 겪게 됩니다. 즉 블로그 포스트는 article요소를, 챕터는 section요소를, 페이지 네비게이션은 nav요소를, 폼 컨트롤 그룹은 fieldset요소를 이용할 수 있습니다.

div요소는 스타일링을 목적으로 한다던가 같은 섹션에 포함된 여러개의 문단을 묶어 비슷한 방법으로 주석을 달고자 할때 유용하게 사용할 수 있습니다.

팁 : div요소는 웹 페이지를 레이아웃하기 위해 CSS와 함께 자주 사용됩니다.

참고 : 기본적으로 브라우저는 항상 div요소 앞뒤에 줄 바꿈을 배치합니다. 이것은 CSS로 제거할 수도 있습니다.

기본 CSS 설정

div {
    display: block;
}

예제

<div style="background-color:lightblue">
    <h3>This is a heading</h3>
    <p>This is a paragraph.</p>
</div>
