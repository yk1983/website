정의

form요소는 웹페이지가 상호작용할 수 있게 하기 위한 form의 최상위 요소로 그 범위를 정의합니다.

form요소는 사용자 입력을위한 HTML 양식을 만드는 데 사용됩니다.

지원 브라우저

속성

Attribute Description
accept 서버가 수락하는(파일 업로드를 통해 제출할 수있는) 파일 유형의 쉼표로 구분 된 목록을 지정합니다. HTML5에서는 지원되지 않습니다.
accept-charset 양식 제출에 사용될 문자 인코딩을 지정합니다.
action 양식 제출시 양식 데이터를 보낼 위치를 지정합니다.
autocomplete 양식에 자동 완성 사용 또는 사용 중지 여부를 지정합니다.
enctype 양식 데이터를 서버로 전송할 때 인코딩 할 방법을 지정합니다. ("method"속성의 값이 "post"일 경우만 해당)
method 양식 데이터를 보낼 때 사용할 HTTP 메소드를 지정합니다.
name 양식의 이름을 지정합니다.
novalidate 제출할 때 양식의 유효성을 검사하지 않도록 지정합니다.
target 양식을 제출 한 후 수신 된 응답을 표시 할 위치를 지정합니다.
  • _blank
  • _parent
  • _self
  • _top

설명

form요소는 다음 형식 요소 중 하나 이상을 포함 할 수 있습니다.

  • input
  • textarea
  • button
  • select
  • options
  • optgroup
  • fieldset
  • label

HTML5에는 autocomplete 및 novalidate의 두 가지 새로운 속성이 추가되었으며 accept 속성이 제거되었습니다.

XHTML에서 name속성은 더 이상 사용되지 않습니다. 대신 global id 속성을 사용하십시오.

기본 CSS 설정

form {
    display: block;
    margin-top: 0em;
}

예제

<form action="/action_page.php" method="get">
    First name: <input type="text" name="fname"><br>
    Last name: <input type="text" name="lname"><br>
    <input type="submit" value="Submit">
</form>
