정의

blockquote요소는 텍스트를 인용문으로 정의한다.

속성

Attribute Description
cite 인용문의 출처를 지정합니다.

설명

blockquote요소는 다른 소스에서 인용된 섹션을 지정합니다.

브라우저는 일반적으로 blockquote요소를 들여 쓰기합니다.

기본 CSS 설정

blockquote {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 40px;
    margin-right: 40px;
}

예제

cite속성을 이용하여 인용된 정보의 URL를 명시

<blockquote cite="http://www.worldwildlife.org/who/index.html">
    인용된 정보의 URL를 명시한다
</blockquote>

서적에서 인용된 정보라면 cite속성에 ISBN을 지정

<blockquote cite="urn:isbn:15215212552">
    인용된 정보의 ISBN 명시한다
</blockquote>

인용된 정보의 타이틀을 명시

<blockquote cite="http://www.worldwildlife.org/who/index.html" title="웹표준">
    인용된 정보의 타이틀을 명시한다
</blockquote>
