정의

button요소는 클릭 가능한 버튼을 정의합니다.

input요소로 생성 된 버튼의 차이점은 button요소 안에는 텍스트나 이미지와 같은 내용을 넣을 수 있습니다.

속성

Attribute Description
autofocus 페이지가 로드될 때 버튼에 자동으로 포커스가 지정되도록 지정합니다.
disabled 버튼을 비활성화해야 함을 나타냅니다.
form 버튼이 속한 하나 이상의 양식을 지정합니다.
formaction 양식을 제출할 때 양식 데이터를 보낼 위치를 지정합니다. type="submit"이어야 합니다.
formenctype 양식 데이터를 서버로 보내기 전에 인코딩 할 방법을 지정합니다. type="submit"이어야 합니다.
formmethod 양식 데이터를 보내는 방법(사용할 HTTP 방법)을 지정합니다. type="submit"이어야 합니다.
formnovalidate 제출시 양식 데이터의 유효성을 검사하지 않도록 지정합니다. type="submit"이어야 합니다.
formtarget 양식을 제출한 후 응답을 표시 할 위치를 지정합니다. type="submit"이어야 합니다.
name 버튼의 이름을 지정합니다.
type 버튼의 유형을 지정합니다.
  • submit: 폼을 제출합니다.
  • reset : 폼을 리셋합니다.
  • button : 지정된 스크립트가 없다면 아무 동작도 하지 않는 버튼입니다.
value 버튼의 초기 값을 지정합니다.

설명

input요소로 생성하는 버튼과 같지만, 이미지나 텍스트등을 포함하고 있어 유연한 디자인이 가능합니다.

버튼의 폼요소의 내부에 버튼을 생성하고 type를 지정하지 않으면 submit로 동작합니다.

팁 : 항상 <button> 요소의 type 속성을 지정하십시오. 다른 브라우저는 <button> 요소에 대해 다른 기본 유형을 사용합니다.

예제

<button type="button">클릭하세요!</button>
