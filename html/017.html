정의

button요소는 버튼을 생성한다.

속성

  • autofocus : 페이지를 로드할때 해당 요소가 자동으로 포커스를 받도록 지정
  • disabled : 포커스, 선택, 변경 등의 조작이 불가능하게 하고 데이터도 서버에 전송하지 않는다. boolean 속성(true와 false로 지정되는 속성)
  • name="이름" : 버튼의 식별자를 지정
  • type=버튼의 종류를 지정
    • submit: 폼을 제출합니다.
    • reset : 폼을 리셋
    • button : 지정된 스크립트가 없다면 아무 동작도 하지 않는 버튼
  • value="값" : 전송되는 데이터의 초기값 지정

설명

input요소로 생성하는 버튼과 같지만, 이미지나 텍스트등을 포함하고 있어 유연한 디자인이 가능하다.

버튼의 폼요소의 내부에 버튼을 생성하고 type를 지정하지 않으면 submit로 동작한다.
