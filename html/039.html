정의

figcaption요소는 부모요소인 figure요소의 내용들에 대한 캡션, 혹은 제목을 나타냅니다.

지원 브라우저

6.0 9.0 4.0 5.0 11.1

설명

figure요소안에서 맨 처음 혹은 맨 마지막에만 올 수 있습니다.

기본 CSS 설정

figcaption { 
    display: block;
}

예제

figure요소를 사용하여 문서의 사진을 마크 업하며, figure요소에는 figcaption도 포함 :
<figure>
    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>
