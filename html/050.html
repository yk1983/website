정의

hr요소는 문단레벨에서 주제의 분리를 나타냅니다 .

지원 브라우저

설명

hr요소는 HTML 페이지에서 주제별 구분을 정의합니다. (예 : 주제 변경).

hr요소는 HTML 페이지에서 내용을 분리하거나 변경을 정의하는 데 사용됩니다.

HTML4.01에서 hr요소는 가로 규칙을 나타냅니다. 그러나 hr요소는 시각적 브라우저에서 여전히 수평 규칙으로 표시 될 수 있지만 이제는 표현 용어가 아닌 의미 용어로 정의됩니다.

모든 레이아웃 속성은 HTML5에서 제거됩니다. 대신 CSS를 사용하십시오.

속성

Attribute Description
align 요소의 정렬을 지정합니다. HTML5에서는 지원되지 않습니다.
noshade 요소가 음영 처리 된 색상 대신 단색 (비 음영)으로 렌더링되도록 지정합니다. HTML5에서는 지원되지 않습니다.
size 요소의 높이를 지정합니다. HTML5에서는 지원되지 않습니다.
width 요소의 너비를 지정합니다. HTML5에서는 지원되지 않습니다.

기본 CSS 설정

hr { 
    display: block;
    margin-top: 0.5em;
    margin-bottom: 0.5em;
    margin-left: auto;
    margin-right: auto;
    border-style: inset;
    border-width: 1px;
}

예제

<h1>HTML</h1>
<p>HTML is a language for describing web pages.....</p>

<hr>

<h1>CSS</h1>
<p>CSS defines how to display HTML elements.....</p>
