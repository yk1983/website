정의

head요소는 문서에 사용되는 메타데이터 집합을 나타냅니다.

head요소는 모든 head요소에 대한 컨테이너입니다.

지원 브라우저

속성

Attribute Description
profile 일련의 규칙이 포함 된 문서의 URL을 지정합니다. <meta> 태그의 content 속성에있는 정보를 명확하게 이해하기 위해 브라우저에서 규칙을 읽을 수 있습니다. HTML5에서는 지원되지 않습니다.

설명

HTML문서는 머리 부분(<head> ... </head>)과 본문 부분(<body> ... </body>)으로 나눌 수 있는데 머리 부분에 해당합니다.

head요소는 문서, 스크립트, 스타일, 메타 정보 등에 대한 제목을 포함 할 수 있습니다.

다음 요소는 head요소 안에 들어갈 수 있습니다.

  • title(이 요소는 HTML 문서에서 필요합니다.)
  • 스타일
  • base
  • link
  • meta
  • script
  • noscript

HTML4.01에서는 head요소가 필요합니다. HTML5에서는 head요소를 생략 할 수 있습니다.

기본 CSS 설정

head { 
    display: none;
}

예제

<!DOCTYPE html>
<html>
<head>
    <title>Title of the document</title>
</head>

<body>
The content of the document......
</body>

</html>
