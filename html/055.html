정의

input쇼tr소는 폼 안에 기본적인 컨트롤을 생성합니다.

지원 브라우저

설명

input 요소는 사용자가 데이터를 입력 할 수있는 입력 필드를 지정합니다.

input 요소는 사용자가 데이터를 입력 할 수있는 입력 컨트롤을 선언하기 위해 form요소 내에서 사용됩니다.

입력 필드는 type 속성에 따라 여러 가지 방법으로 변화 할 수 있습니다.

참고 : input 요소는 비어 있으며 속성 만 포함합니다.

팁 : input 요소의 레이블을 정의 하려면 label 요소를 사용하십시오.

HTML5에서는 align속성이 지원되지 않습니다.

HTML5에서 input 요소에는 몇 가지 새로운 속성이 있고 type 속성에는 몇 가지 새로운 값이 있습니다.

HTML에서 input 요소에는 닫는 태그가 없습니다.

XHTML에서 input 요소는 <input />과 같이 올바르게 닫아야합니다.

속성

Attribute Description
accept 서버가 승인하는 파일 유형을 지정합니다. (type="file"인 경우에만 해당)
align 이미지 입력의 정렬을 지정합니다. (type="image" 경우에만 해당) HTML5에서는 지원되지 않습니다.
alt 이미지의 대체 텍스트를 지정합니다. (type="image" 경우에만 해당)
autocomplete input 요소가 자동 완성을 사용할 수 있는지 여부를 지정합니다.
autofocus 페이지가 로드 될 때 input 요소가 자동으로 포커스를 가져 오도록 지정합니다.
checked 페이지가로드 될 때 input 요소가 미리 선택되도록 지정합니다. (type="checkbox" 또는 type="radio"의 경우)
dirname 텍스트 방향을 제출하도록 지정합니다.
disabled input 요소가 비활성화되도록 지정합니다.
form input 요소가 속한 하나 이상의 양식을 지정합니다.
formaction 양식을 제출할 때 입력 제어를 처리 할 파일의 URL을 지정합니다. (type="submit" 및 type="image"의 경우)
formenctype 양식 데이터를 서버로 전송할 때 인코딩 할 방법을 지정합니다. (type="submit" 및 type="image"의 경우)
formmethod 작업 URL로 데이터를 보내는 HTTP 메서드를 정의합니다. (type="submit" 및 type="image"의 경우)
formnovalidate 폼 요소를 제출할 때 유효성을 검증해서는 안됨을 정의합니다.
formtarget 양식을 제출 한 후 수신되는 응답을 표시 할 위치를 지정합니다. (type="submit" 및 type="image"의 경우)
height input 요소의 높이를 지정합니다. (type="image" 경우에만 해당)
width input 요소의 너비를 지정합니다. (type="image" 경우에만 해당)
list input 요소에 대해 미리 정의 된 옵션을 포함하는 datalist 요소를 참조합니다.
max input 요소의 최대 값을 지정합니다.
maxlength input 요소에 허용되는 최대 문자 수를 지정합니다.
min input 요소의 최소값을 지정합니다.
multiple 사용자가 input 요소에 둘 이상의 값을 입력 할 수 있도록 지정합니다.
name input 요소의 이름을 지정합니다.
pattern input 요소의 값을 검사하는 정규식을 지정합니다.
placeholder input 요소의 예상 값을 설명하는 짧은 힌트를 지정합니다.
readonly 입력 필드가 읽기 전용임을 지정합니다.
required 양식을 제출하기 전에 입력 필드를 작성하도록 지정합니다. (빈 값으로 작성되지 않도록 유효성을 검증합니다.)
sizes input 요소의 폭을 문자로 지정합니다.
src 제출 단추로 사용할 이미지의 URL을 지정합니다. (type="image" 경우에만 해당)
step 입력 필드의 유효한 번호 간격을 지정합니다.
type 표시 할 유형 input 요소를 지정합니다.
