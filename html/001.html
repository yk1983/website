정의

a 요소는 한 페이지에서 다른 페이지로 연결하는 데 사용되는 하이퍼 링크를 정의합니다.

a 요소의 가장 중요한 속성은 링크의 목적지를 나타내는 href 속성입니다.

기본적으로 링크는 모든 브라우저에서 다음과 같이 나타납니다.

  • 방문하지 않은 링크에는 밑줄이 그어져 파란색으로 표시됩니다.
  • 방문한 링크에 밑줄이 그어지고 자주색입니다.
  • 활성 링크에는 밑줄이 그어지고 빨간색으로 표시됩니다.

지원 브라우저

속성

Attribute Description
charset 링크 된 문서의 문자 집합을 지정합니다. HTML5에서는 지원되지 않습니다.
coords 링크의 좌표를 지정합니다. HTML5에서는 지원되지 않습니다.
download 사용자가 하이퍼 링크를 클릭 할 때 대상을 다운로드하도록 지정합니다.
href URL 링크가 연결되는 페이지의 URL을 지정합니다.
hreflang 링크 된 문서의 언어를 지정합니다.
media 링크 된 문서가 최적화되어있는 미디어 / 장치를 지정합니다.
name 앵커의 이름을 지정합니다. HTML5에서는 지원되지 않습니다. 대신 global id 속성을 사용하십시오.
ping 링크를 따라 가면 본문 핑이있는 게시물 요청이 브라우저에서 백그라운드로 전송되는 URL의 공백으로 구분 된 목록을 지정합니다. 일반적으로 추적에 사용됩니다.
rel 현재 문서와 링크 된 문서 간의 관계를 지정합니다.
rev 링크 된 문서와 현재 문서 간의 관계를 지정합니다. HTML5에서는 지원되지 않습니다.
shape HTML5에서는 지원되지 않습니다. 링크 모양을 지정합니다.
target 링크 된 문서를 열 위치를 지정합니다.
  • _blank
  • _parent
  • _self
  • _top
  • framename
type 링크 된 문서의 미디어 유형을 지정합니다.

설명

a 요소는 문단, 목록, 테이블뿐만 아니라 섹션 전체를 감쌀 수도 있다.

a 요소안에 버튼이나 링크등 인터렉티브한 요소가 포함되지 않도록 한다.

링크의 상태

스타일을 정의할 때 링크 상태들은 순서대로 정의 해야 의도한 대로 스타일을 적용할 수 있다. active를 hover보다 먼저 정의 했다면 hover가 active를 오버라이드 하여 active상태를 볼 수가 없게 된다.

  • link : 기본적인 링크 상태로 아직 방문한적이 없거나 아무런 행동도 하기전인 상태
  • visited : 방문한 적이 있는 링크의 상태
  • hover : 마우스 커서가 링크위에 올라가 있는 상태
  • focus : 링크에 키보드등을 이용해서 포커스가 위치해있는 상태
  • active : 마우스를 눌렀다가 놓는 동안의 상태

기본 CSS 설정

a:link, a:visited { 
    color: (internal value);
    text-decoration: underline;
    cursor: auto;
}

a:link:active, a:visited:active { 
    color: (internal value);
}

예제

하이퍼 링크 만들기 하이퍼 링크를 만드는 방법.
<a href="https://www.w3schools.com">Visit W3Schools.com!</a>

클릭 가능한 이미지 만들기 이미지를 링크로 사용하는 방법.
<img border="0" alt="W3Schools" src="img/logo_w3s.gif" width="100" height="100">

새 브라우저 창에서 링크 열기
방문자가 웹 사이트를 떠나지 않아도되도록 새 브라우저 창에서 링크를 여는 방법.
<a href="https://www.w3schools.com" target="_blank">Visit W3Schools.com!</a>

전자 메일 링크 만들기
메일 메시지에 연결 하는 방법 (메일이 설치된 경우에만 작동 함).
<a href="mailto:yong1983@naver.com?Subject=Hello%20again" target="_top">Send Mail</a>

다른 전자 메일 링크 만들기 다른 mailto 링크.
<a href="mailto:yong1983@naver.com?cc=someoneelse@example.com&bcc=andsomeoneelse@example.com&subject=Summer%20Party&body=You%20are%20invited%20to%20a%20big%20summer%20party!" target="_top">Send mail!</a>

전화 링크 만들기 전화 번호 링크 방법.
<a href="tel:+4733378901">+47 333 78 901</a>

동일한 페이지의 다른 섹션에
연결하기 페이지 내의 지정된 ID를 가진 요소에 연결하는 방법 (name 속성은 HTML5에서 지원되지 않습니다).
<a href="#top">Go to top</a>

JavaScript에 연결하기
href 속성 안에 JavaScript를 추가하는 법.
<a href="javascript:alert('Hello World!');">Execute JavaScript</a>
