정의

col요소는 colgroup요소 내의 각 열에 대한 열 속성을 지정합니다.

col요소는 각 행에 대해 각 셀의 스타일을 반복하는 대신 전체 열에 스타일을 적용하는데 유용합니다.

지원 브라우저

속성

Attribute Description
align HTML5에서는 지원되지 않습니다. col요소와 관련된 내용의 정렬을 지정합니다.
char HTML5에서는 지원되지 않습니다. col요소와 관련된 내용의 문자에 대한 정렬을 지정합니다.
charoff HTML5에서는 지원되지 않습니다. char 속성에 의해 지정된 문자로부터 내용이 정렬 될 문자의 수를 지정합니다.
span col요소가 확장되어야하는 열의 수를 지정합니다.
valign HTML5에서는 지원되지 않습니다. col요소와 관련된 내용의 세로 맞춤을 지정합니다.
width HTML5에서는 지원되지 않습니다. col요소의 너비를 지정합니다.

설명

HTML 4.01의 대부분의 속성은 HTML5에서 지원되지 않습니다.

caption요소 바로 뒤에 thead요소와 tfoot요소, tbody요소 이전에 기술합니다.

기본 CSS 설정

col { 
    display: table-column;
}

예제

<table>
    <colgroup>
        <col span="2" style="background-color:red">
        <col style="background-color:yellow">
    </colgroup>
    <tr>
        <th>ISBN</th>
        <th>Title</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>3476896</td>
        <td>My first HTML</td>
        <td>$53</td>
    </tr>
</table>
