정의

fieldset요소는 양식의 관련 요소를 그룹화하는 데 사용됩니다.

fieldset요소는 관련 요소 주위에 상자를 그립니다.

지원 브라우저

속성

Attribute Description
disabled 관련 양식 요소 그룹을 비활성화해야 함을 지정합니다.
form 필드 세트가 속한 하나 이상의 양식을 지정합니다.
name 필드 세트의 이름을 지정합니다.

설명

팁 : legend요소를 이용하여 캡션을 부여할 수 있습니다.

중첩하여 사용할 수도 있습니다.

기본 CSS 설정

fieldset { 
    display: block;
    margin-left: 2px;
    margin-right: 2px;
    padding-top: 0.35em;
    padding-bottom: 0.625em;
    padding-left: 0.75em;
    padding-right: 0.75em;
    border: 2px groove (internal value);
}

예제

<form>
    <fieldset>
        <legend>Personalia:</legend>
        Name: <input type="text"><br>
        Email: <input type="text"><br>
        Date of birth: <input type="text">
    </fieldset>
</form>
