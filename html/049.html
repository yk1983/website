정의

hgroup요소는 섹션의 제목그룹을 나타냅니다.

hgroup은 w3c 표준에서 사라졌으나 whatwg 표준에 아직 존재합니다.

지원 브라우저

설명

이 요소는 제목이 여러 레벨을 가질때, 즉 부제목이나 대체타이틀이나 태그라인을 가질때 h1~h6요소들을 그룹짓기위해 사용됩니다.

기본 CSS 설정

hgroup { 
    display: block;
}

예제

<hgroup>
    <h1>Most important heading here</h1>
    <h3>(Less important heading here)</h3>
</hgroup>
