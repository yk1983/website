정의

del요소는 문서에서 삭제 된 텍스트를 정의합니다.

지원 브라우저

속성

Attribute Description
cite del요소의 변화에 대해 설명이 되어 있는 곳의 위치를 명시합니다.
datetime 삭제한 시간을 지정합니다.

설명

팁 : 삽입 된 텍스트를 마크 업 하려면 ins요소를 사용하십시오.

팁 : 문서에서 업데이트 및 수정 사항을 마크 업하려면 del요소 및 ins요소를 사용하십시오. 브라우저는 일반적으로 삭제 된 텍스트를 통해 줄을 긋고 삽입 된 텍스트에 밑줄을 긋습니다.

기본 CSS 설정

del {
    text-decoration: line-through;
}

예제

<p>My favorite color is <del>blue</del> <ins>red</ins>!</p>
