정의

dt요소는 dl요소의 용어-설명 그룹에서 용어나 이름을 나타냅니다.

지원 브라우저

설명

dl요소의 하부 요소입니다.

용어를 설명하는 dd요소와 함께 사용됩니다.

dt요소는 dl(설명 목록 정의) 및 dd(각 용어 / 이름 설명) 와 함께 사용됩니다.

HTML 4.01에서 정의 목록의 항목을 정의합니다. HTML5에서는 설명 목록에 용어 / 이름을 정의합니다.

기본 CSS 설정

dt { 
    display: block;
}

예제

<dl>
    <dt>Coffee</dt>
    <dd>Black hot drink</dd>
    <dt>Milk</dt>
    <dd>White cold drink</dd>
</dl>
