정의

cite요소는 작품 제목(예 : 책, 노래, 영화, TV 프로그램, 그림, 조각품 등)을 정의합니다.

지원 브라우저

설명

참고 : 사람의 이름은 작품의 제목이 아닙니다. 따라서 cite요소에 작품의 제목 이상의 것을 포함해서는 안됩니다.

HTML 4.01에서 인용문을 정의하였고, HTML5에서는 저작의 제목을 정의합니다.

cite요소는 인용문과는 구별되므로 반드시 제목을 마크업하는 용도로만 사용해야 합니다.

사람의 이름이 작품과 관련있다고 하더라도 cite로 마크업해서는 안됩니다. 가령 사람 이름은 b요소나 span요소를 사용하도록합니다.

기본 CSS 설정

cite { 
    font-style: italic;
}

예제

<cite>The Scream</cite> by Edward Munch. Painted in 1893.
