정의

dialog요소는 대화 상자 또는 창을 정의합니다.

지원 브라우저

37.0 N/S 59.0 6.0 24.0

속성

Attribute Description
open 대화 요소가 활성화되어 있고 사용자가 대화 상자 요소와 상호 작용할 수 있도록 지정합니다.

설명

dailog요소를 사용하면 웹 페이지에서 팝업 대화 상자 및 모달을 쉽게 만들 수 있습니다.

예제

<table>
    <tr>
        <th>January <dialog open>This is an open dialog window</dialog></th>
        <th>February</th>
        <th>March</th>
    </tr>
    <tr>
        <td>31</td>
        <td>28</td>
        <td>31</td>
    </tr>
</table>
