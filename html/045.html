정의

frameset요소는 프레임 세트를 정의합니다.

HTML5에서 지원되지 않습니다.

지원 브라우저

속성

Attribute Description
cols 프레임 세트의 열 수와 크기를 지정합니다. HTML5에서는 지원되지 않습니다.
rows 프레임 셋의 행 수와 크기를 지정합니다. HTML5에서는 지원되지 않습니다.

설명

frameset요소는 하나 이상의 frame요소를 포함합니다. 각 frame요소는 별도의 문서를 보유 할 수 있습니다.

frameset요소는 얼마나 많은 열이나 행을 프레임 셋에 넣을 것인지를 지정하고, 공간의 비율 / 픽셀 당 얼마나 많은 공간이 각각을 차지할 것인지를 지정합니다.

참고 : 프레임이 포함 된 페이지의 유효성을 검사하려면 <!DOCTYPE> 이 "HTML Frameset DTD"또는 "XHTML Frameset DTD"로 설정되어 있는지 확인하십시오.

예제

<frameset cols="25%,50%,25%">
    <frame src="frame_a.htm">
    <frame src="frame_b.htm">
    <frame src="frame_c.htm">
</frameset>
