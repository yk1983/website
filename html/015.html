정의

body요소는 문서의 본문을 정의합니다.

body요소는 텍스트, 하이퍼 링크, 이미지, 표, 목록 등과 같은 HTML 문서의 모든 내용을 포함합니다.

속성

Attribute Description
alink HTML5에서는 지원되지 않습니다. 문서의 활성 링크의 색상을 지정합니다.
background HTML5에서는 지원되지 않습니다. 문서의 배경 이미지를 지정합니다.
bgcolor HTML5에서는 지원되지 않습니다. 문서의 배경색을 지정합니다.
link HTML5에서는 지원되지 않습니다. 문서에서 방문하지 않은 링크의 색상을 지정합니다.
text HTML5에서는 지원되지 않습니다. 문서의 텍스트 색상을 지정합니다.
vlink HTML5에서는 지원되지 않습니다. 문서에서 방문한 링크의 색상을 지정합니다.

설명

HTML문서는 머리 부분(<head> ... </head>)과 본문 부분(<body> ... </body>)으로 나눌 수 있는데 본문 부분에 해당합니다.

올바른 문서에서는 하나의 body요소만이 존재합니다.

기본 CSS 설정

body { 
    display: block;
    margin: 8px;
}

body:focus { 
    outline: none;
}

예제

<!DOCTYPE html>
<html>
<head>
<title>문서 제목</title>
</head>

<body>
문서 내용 ...
</body>

</html>
