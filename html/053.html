정의

iframe요소는 문서내에 다른 문서를 내포할 수 있게 해줍니다.

지원 브라우저

설명

iframe요소는 인라인 프레임을 지정합니다.

인라인 프레임은 현재 HTML 문서 내에 다른 문서를 삽입하는데 사용됩니다.

팁 : iframe을 지원하지 않는 브라우저를 처리하려면 여는 iframe 태그와 닫는 iframe 태그 사이에 텍스트를 추가하여 사용자에게 알리십시오.

팁 : CSS를 사용하여 iframe의 스타일을 지정하십시오. (스크롤바를 포함하기까지)

HTML5에는 몇 가지 새로운 속성이 추가되었으며 HTML5에는 여러 HTML4.01 속성이 제거되었습니다.

XHTML에서 name 속성은 더 이상 사용되지 않으며 제거됩니다. 대신 글로벌 id 속성을 사용합니다.

속성

Attribute Description
align 주변 요소에 따라 iframe의 정렬을 지정합니다. HTML5에서는 지원되지 않습니다.
frameborder iframe 주위에 테두리를 표시할지 여부를 지정합니다. HTML5에서는 지원되지 않습니다.
height iframe의 높이를 지정합니다.
width iframe의 너비를 지정합니다.
longdesc iframe의 내용에 대한 자세한 설명이 포함 된 페이지를 지정합니다. HTML5에서는 지원되지 않습니다.
marginheight iframe 요소의 내용의 위쪽 및 아래쪽 여백을 지정합니다. HTML5에서는 지원되지 않습니다.
