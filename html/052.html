정의

i요소는 텍스트를 이탤릭체로 장식하는 스타일적인 요소입니다.

지원 브라우저

설명

i 요소를 사용한 의미를 class 속성을 이용하여 나타낼 수 있습니다.

i 요소보다 더 적합한 요소가 있는지 고려해보길 권장합니다. 예를 들어 강조를 하기위한 목적이라면 em요소를 사용하고, 정의를 목적으로 하고 있다면 dfn요소를 사용하길 권합니다.

i 요소는 대체 음성 또는 분위기의 텍스트 일부를 정의합니다. 내용은 일반적으로 기울임 꼴로 표시됩니다.

i 요소는 전문 용어, 다른 언어의 구문, 생각 또는 배 이름 등을 나타내는 데 사용할 수 있습니다.

다음에서 더 적절한 의미 요소가없는 경우에만 i 요소를 사용하십시오.

  • <em> (강조 텍스트)
  • <strong> (중요 텍스트)
  • <mark> (강조 표시된 텍스트로 표시)
  • <cite> (작품의 제목)
  • <dfn> (정의 용어)

HTML4.01에서 i 요소는 기울임 꼴로 텍스트를 렌더링하는 데 사용되었습니다. 그러나 HTML5의 경우에는 그렇지 않습니다. 기울임 꼴이 목적이라면 스타일 시트를 사용하여 i 요소 내부의 텍스트 서식을 지정할 수 있습니다.

기본 CSS 설정

i { 
    font-style: italic;
}

예제

<p>He named his car <i>The lightning</i>, because it was very fast.</p>
