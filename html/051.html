정의

html요소는 HTML문서의 루트에 해당하며, 문서의 기본 요소입니다.

지원 브라우저

설명

전체 문서는 <html> ... &l/html>안에 기술됩니다.

html요소는 head요소와 body요소를 포함합니다.

하나의 문서에 하나만 존재할 수 있습니다.

html요소는 HTML 문서임을 브라우저에 알립니다. (또한 문서의 버전정보를 브라우저에 알려줍니다.)

xmlns속성은 XHTML에서는 필요하지만 HTML에서는 유효하지 않습니다.

속성

Attribute Description
xmlns XML 네임 스페이스 특성을 지정합니다. (콘텐츠가 XHTML을 준수해야하는 경우)

기본 CSS 설정

html { 
    display: block;
}

html:focus { 
    outline: none;
}

예제

<!DOCTYPE HTML>
<html>
<head>
<title>Title of the document</title>
</head>

<body>
The content of the document......
</body>

</html>
