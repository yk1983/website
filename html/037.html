정의

embed요소는 브라우저에 설치되어 있는 플러그인이 필요한 외부 어플리케이션을 동작시킵니다.

지원 브라우저

속성

Attribute Description
height 포함 된 콘텐츠의 높이를 지정합니다.
src 임베드할 외부 파일의 주소를 지정합니다.
type 포함 된 컨텐츠의 미디어 유형을 지정합니다.
width 삽입 된 내용의 너비를 지정합니다.

설명

embed HTML5에서 새로 추가되었습니다.

참고 : 많은 웹 브라우저가 오랫동안 embed요소를 지원했습니다. 그러나 HTML4 사양에 포함되지 않았습니다. HTML5에서 새로 추가되었으며 HTML5 페이지에서 유효성을 검사합니다. 그러나 HTML4 페이지에서 이 페이지를 사용하면 페이지가 유효하지 않습니다.

기본 CSS 설정

embed:focus { 
    outline: none;
}

예제

<embed src="helloworld.swf">
