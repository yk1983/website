정의

audio요소는 음악 또는 기타 오디오 스트림과 같은 사운드를 정의합니다.

현재 audio요소에는 MP3, WAV 및 OGG의 3가지 파일 형식이 있습니다.

지원 브라우저

Browser MP3 WAV OGG
9.0 YES NO NO
4.0 YES YES YES
3.5 YES YES YES
4.0 YES YES NO
10.5 YES YES YES

속성

Attribute Description
autoplay 오디오가 준비되는 즉시 재생을 시작하도록 지정합니다.
controls 오디오 컨트롤이 표시되도록 지정합니다. (예 : 재생 / 일시 중지 단추 등)
loop 오디오가 끝날 때마다 다시 시작하도록 지정합니다.
muted 오디오 출력을 음소거하도록 지정합니다.
preload 제작자가 페이지 로드시 오디오를 로드해야 한다고 생각하는지 여부 및 그 방법을 지정합니다. (다운로드 상태 : none / metadata / auto)
src 오디오 파일의 URL을 지정합니다.

설명

브라우저가 audio요소에서 제공하고 있는 오디오를 재생하지 못하는 경우 audio요소 내부에 있는 내용을 출력합니다.

예제

<audio controls>
    <source src="horse.ogg" type="audio/ogg">
    <source src="horse.mp3" type="audio/mpeg">
    Your browser does not support the audio tag.
</audio>
