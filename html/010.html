정의

base요소는 HTML문서의 기준 URL을 명시한다.

속성

  • href : 문서의 기준이 되는 URL을 지정
  • target : 링크를 어떤 프레임에 열것인지를 결정
    • _blank : 새창에 열기
    • _parent : 부모창에 열기
    • _self : 자신의 창에 열기
    • _top : 화면이 여러 프레임으로 이루어져있는 경우 모든 프레임을 지우고 전체 화면에 열기
    • [name] : 지정된 이름에 링크를 열기

설명

문서의 head요소 안에서 사용

base는 문서 당 하나만 지정 가능

href 속성이나 target 속성중 반드시 하나 이상을 써야하며 둘 다 사용가능

예제

<!DOCTYPE html>
<html>
<head>
<base href="https://www.w3schools.com/images/" target="_blank">
</head>
<body>

<p><img src="stickman.gif" width="24" height="39" alt="Stickman"> 이미지의 상대 주소만 지정했지만 head섹션에 base URL을 지정하였으므로 브라우저에서는 "https://www.w3schools.com/images/stickman.gif"에서 이미지를 검색한다.</p>

<p><a href="https://www.w3schools.com">W3Schools</a> target="_blank" 속성을 지정하지 않았더라도 링크가 새 창에서 열린다. 이는 base요소의 대상이 "_blank"로 설정되어 있기 때문이다.</p>

</body>
</html>
