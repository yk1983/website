NetworkEx4.java

URL을 연결하여 내용을 읽어오는 예제입니다. 만약 URL이 유효하지 않으면 Malformed- URLException이 발생합니다. 읽어올 데이터가 문자데이터이기 때문에 BufferedReader을 사용하였으며, openStream()을 호출해서 URL의 InputStream을 얻은 이후로는 파일로 부터 데이터를 읽는 것과 다르지 않습니다.
openStream()은 openConnection()을 호출해서 HttpURLConnection을 얻은 다음 여기에 다시 getInputStream()을 호출한 것과 같습니다. 즉, URL에 연결해서 InputStream을 얻어옵니다.

NetworkEx4.java

package chapter16;

import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.net.URL;

public class NetworkEx4 {
	public static void main(String args[]) {
		URL url = null;
		BufferedReader input = null;
		String address = "https://www.w3schools.com/html/tryit.asp?filename=tryhtml_form_submit#index";
		String line = "";

		try {
			url = new URL(address);

			input = new BufferedReader(new InputStreamReader(url.openStream()));

			while ((line = input.readLine()) != null) {
				System.out.println(line);
			}
			input.close();
		} catch (Exception e) {
			e.printStackTrace();
		}
	}
}
