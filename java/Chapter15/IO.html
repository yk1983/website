Chapter 15. 입출력(I/O)

I/O는 Input과 Output의 약자입니다. 간단히 줄여서 입출력이라고 하며, 입출력은 컴퓨터 내부 또는 장치와 프로그램간의 데이터를 주고 받는 것을 말합니다. 지금까지 사용한 Scanner를 이용하여 키보드로 데이터를 입력받는다거나 System.out.println을 이용하여 화면에 출력하는 것이 가장 기본적인 입출력의 예입니다.
Chapter 15. 입출력(I/O)에서 배울 내용은 다음과 같습니다.

  • 스트림 (stream)
  • 바이트기반 스트림
  • 문자기반 스트림
  • 표준입출력
  • File
  • 직렬화 (Serialization)
