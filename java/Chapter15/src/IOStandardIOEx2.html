StandardIOEx2.java

System.out, System.err 모두 출력대상이 콘솔이기 때문에 동일한 결과를 얻습니다. (단, System.err은 이클립스 콘솔창에서 텍스트가 빨간색으로 표시됩니다.)

소스코드

package std.java.chapter15;

public class StandardIOEx2 {
	public static void main(String[] args) {
		System.out.println("out : Hello World!");
		System.err.println("err : Hello World!");
	}
}

실행결과

out : Hello World!
err : Hello World!
