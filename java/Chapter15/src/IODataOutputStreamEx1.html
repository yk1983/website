DataOutputStreamEx1.java

FileOutputStream을 기반으로 하는 DataOuputStream을 생성한 후, DataOutputStream의 메서드들을 이용해서 sample.dat파일에 값들을 출력하는 예제입니다.

소스코드

package std.java.chapter15;

import java.io.DataOutputStream;
import java.io.FileOutputStream;
import java.io.IOException;

public class DataOutputStreamEx1 {
	public static void main(String args[]) {
		// DataOutputStream 객체를 생성
		FileOutputStream fos = null;
		DataOutputStream dos = null;

		try {
			fos = new FileOutputStream("src/std/java/chapter15/sample.dat");
			dos = new DataOutputStream(fos);

			// sample.dat파일에 값들을 출력
			dos.writeInt(10);
			dos.writeFloat(20.0f);
			dos.writeBoolean(true);

			dos.close();
		} catch (IOException e) {
			e.printStackTrace();
		}
	}
}
출력한 값들은 이진 데이터(binary data)로 저장됩니다.
문자 데이터(text data)가 아니므로 문서 편집기로 열어봐도 알 수 없는 글자들로 이루어져 있습니다. (16진수 코드로 볼수있는 에디터에서 확인할 수 있음)
