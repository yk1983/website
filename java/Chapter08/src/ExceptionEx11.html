ExceptionEx11.java

메서드의 throws를 명시하고 자신을 호출한 메서드에 예외처리를 전달하여 main메서드에서 예외처리하는 예제

  1. 예외가 발생했을 때, 모두 3개의 메서드(main, myMethod1, myMethod2)가 호출스택에 있었으며,
  2. 예외가 발생한 곳은 제일 윗줄에 있는 myMethod2()라는 것과
  3. main메서드가 myMethod1()을 그리고 myMethod1()은 myMethod2()를 호출했다는 것을 알 수 있습니다.

소스코드

package kinco.java.std;

public class ExceptionEx11 {
	public static void main(String[] args) {
		try {
			myMethod1();
		} catch (Exception e) {
			System.out.println("main메서드에서 예외가 처리되었습니다.");
			e.printStackTrace();
		}
	}

	private static void myMethod1() throws Exception{
		// 이 메서드를 호출한 곳에 예외처리를 전달
		myMethod2();
	}

	private static void myMethod2() throws Exception {
		// 예외를 발생
		// 이 메서드를 호출한 곳에 예외처리를 전달
		throw new Exception();
	}
}
