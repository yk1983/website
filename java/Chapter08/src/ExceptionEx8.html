ExceptionEx8.java

catch블럭의 문장 수행 중에 return문을 만나도 finally블럭의 문장들을 수행하는 예제

소스코드

package kinco.java.std;

public class ExceptionEx8 {
	public static void main(String[] args) {
		myMethod();  // static 메서드이므로 직접 호출
		System.out.println("myMethod()의 수행을 마치고 돌아와 실행합니다.");
	}

	private static void myMethod() {
		try {
			System.out.println("myMethod()가 호출 되었습니다.");
			return;	// 현재 실행 중인 메서드를 종료
		} catch (Exception e) {
			e.printStackTrace();
		} finally {
			System.out.println("myMethod()의 finally 블럭이 실행되었습니다.");
		}
	}
}

실행결과

myMethod()가 호출 되었습니다.
myMethod()의 finally 블럭이 실행되었습니다.
myMethod()의 수행을 마치고 돌아와 실행합니다.
