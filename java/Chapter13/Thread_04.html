스레드 우선순위

스레드는 우선순위(priority)라는 속성(멤버변수)을 가지고 있습니다. 이 우선순위의 값에 따라 스레드가 얻는 실행시간이 달라집니다. 스레드가 수행하는 작업의 중요도에 따라 스레드의 우선순위를 서로 다르게 지정하여 특정 스레드가 더 많은 작업시간을 갖도록 할 수 있습니다.

스레드의 우선순위 지정하기

void setPriority(int newPriority)  // 스레드의 우선순위를 지정한 값으로 변경합니다.
int getPriority()  // 스레드의 우선순위를 반환합니다.

public static final int MAX_PRIORITY = 10  // 최대우선순위
public static final int MIN_PRIORITY = 1   // 최소우선순위
public static final int NORM_PRIORITY = 5  // 보통우선순위

스레드의 우선순위는 스레드를 생성한 스레드로부터 상속받으며, 스레드가 가질 수 있는 우선순위의 범위는 1 ~ 10이며 숫자가 높을수록 우선순위가 높습니다. 스레드의 우선순위는 스레드를 실행하기 전에만 우선순위를 변경할 수 있으며, 스레드의 우선순위가 다르면 우선순위가 높은 스레드에 상대적으로 더 많은 양의 실행시간이 주어지고 결과적으로 우선순위가 높은 스레드가 더 빨리 완료될 수 있습니다.
