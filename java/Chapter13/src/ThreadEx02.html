ThreadEx02

사용자로부터 입력받는 부분과 화면에에 숫자를 출력하는 부분을 두 새의 스레드로 나누어서 처리했기 때문에 사용자가 입력을 마치지 않았어도 화면에 숫자가 출력되는 예제입니다.

ThreadEx02.java

package chapter13;

import javax.swing.JOptionPane;

public class ThreadEx02 {
	public static void main(String[] args) throws Exception {
		ThreadB th1 = new ThreadB();
		th1.start();

		String input = JOptionPane.showInputDialog("아무 값이나 입력하세요.");
		System.out.println("입력하신 값은 " + input + "입니다.");
	}
}

ThreadB.java

package chapter13;

public class ThreadB extends Thread {
	public void run() {
		for(int i=10; i > 0; i--) {
			System.out.println(i);
			try {
				sleep(1000);
			} catch(Exception e ) {}
		}
	}
}
