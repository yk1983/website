배열의 선언과 생성

배열의 선언

배열을 선언하는 방법은 원하는 타입의 변수를 선언하고 변수 또는 타입에 배열임을 의미하는 대괄호[]를 붙이면 됩니다.
타입[] 변수이름; 또는 타입 변수이름[];

배열의 선언방법과 선언 예

선언방법 선언 예
타입[] 변수이름; int[] score;
String[] name;
타입 변수이름[]; int score[];
String name[];

배열의 생성

배열을 선언한 다음에는 배열을 생성해야 합니다. 배열을 생성하기 위해서는 다음과 같이 연산자 new와 함께 배열의 타입과 길이를 지정해주어야 합니다.
new 타입[길이]

배열의 선언과 생성 예

int[] score;    // 배열을 선언 (배열을 다루기 위한 참조변수 선언)
score = new int[5];    // 배열을 생성 (실제 저장공간을 생성)
					
int[] score = new int[5];    // 배열의 선언과 생성을 동시에
