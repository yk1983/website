제약조건

NOT NULL

기본적으로 열은 NULL 값을 포함 할 수 있습니다. NOT NULL 제약 조건은 열이 NULL 값을 허용하지 않도록 강제합니다. 이렇게하면 필드에 항상 값이 포함될 수 있습니다. 즉, 새 레코드를 삽입하거나 업데이트 할 때, 이 필드에 값을 추가하지 않거나 NULL 값으로 레코드를 업데이트 할 수 없습니다.
다음 SQL은 "ID", "LastName"및 "FirstName"열이 NULL 값을 허용하지 않도록합니다.

Example

CREATE TABLE Persons (
    ID int NOT NULL
,   LastName varchar(255) NOT NULL
,   FirstName varchar(255) NOT NULL
,   Age int
);

* 제약조건을 설정할 때 CONSTRAINT 제약조건명을 생략할 수 있습니다. CONSTRAINT를 생략하면 제약조건명이 자동으로 부여됩니다.
