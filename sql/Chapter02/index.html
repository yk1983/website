Chapter 02.
데이터 정의어(DDL)

데이터 정의 언어(Data Definition Language, DDL)은 데이터베이스의 스키마 객체를 생성(CREATE), 변경(ALTER), 제거(DROP) 하거나 권한의 부여나 박탈(GRANT, REVOKE), 주석(COMMENT), 자료의 버림(TRUNCATE) 등을 수행하는 문장의 집단을 의미합니다.
데이터 정의 언어에는 많은 종류의 DDL이 존재하며, PL/SQL 블록에서는 DDL을 사용할 수 없습니다.
데이터 정의 언어에서 문장의 첫 단어 각 문장은 CREATE, ALTER, DROP, TRUNCATE, GRANT, REVOKE, COMMENT 등으로 시작합니다.
