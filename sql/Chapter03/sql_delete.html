DELETE

DELETE 문은 테이블의 기존 레코드를 삭제하는 데 사용됩니다.

DELETE Syntax

DELETE
FROM   table_name
WHERE  condition;

또는

DELETE *
FROM   table_name
WHERE  condition;

UPDATE 문과 마찬가지로 DELETE 문을 사용하는 경우에는 테이블에서 레코드를 삭제 할 때 다음과 같은 부분을 주의해야합니다.
DELETE 문의 WHERE 절을 확인하십시오.
WHERE 절은 삭제될 레코드를 지정합니다. WHERE 절을 생략하면 테이블의 모든 레코드가 삭제됩니다. 삭제 될 레코드 수를 결정하는 것은 WHERE 절입니다.

Example

-- "Customers" 테이블에서 고객 "Alfreds Futterkiste"를 삭제합니다.
DELETE FROM Customers
WHERE  CustomerName = 'Alfreds Futterkiste';

-- WHERE절을 사용하지 않으면 "Customers" 테이블의 모든 고객정보가 삭제됩니다.
DELETE FORM Customers;
