UPDATE - 데이터 수정

UPDATE 문은 테이블의 기존 레코드를 수정하는 데 사용됩니다.

UPDATE 문을 사용하는 경우에는 테이블에서 레코드를 업데이트 할 때 다음과 같은 부분을 주의해야합니다.
UPDATE 문에서 WHERE 절을 확인해야합니다.
WHERE 절은 갱신해야하는 레코드를 지정하는데, WHERE 절을 생략하면 테이블의 모든 레코드가 업데이트됩니다. 업데이트 될 레코드 수를 결정하는 것은 WHERE 절입니다.

Syntax

UPDATE table_name
SET    column1 = value1, column2 = value2, ...
WHERE  condition;
  • 수정될 열(column)은 SET 문 다음에 위치합니다.
  • 값(value)는 수정되어 변경될 값입니다.
  • 각각의 테이터 구분은 콤마(,)로 구분합니다.

Example

-- 첫 번째 고객(CustomerID의 값이 1인 데이터)을 새 담당자 및 새 도시로 업데이트합니다.
UPDATE Customers
SET    ContactName = 'Alfred Schmidt', City= 'Frankfurt'
WHERE  CustomerID = 1;

-- country가 "Mexico"인 모든 레코드에 대해 연락처 이름을 "Juan"으로 업데이트합니다.
UPDATE Customers
SET    ContactName = 'Juan'
WHERE  Country = 'Mexico';

-- WHERE 절을 생략하면 모든 레코드가 업데이트됩니다.
UPDATE Customers
SET    ContactName = 'Juan';
